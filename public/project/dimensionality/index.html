Dimensionality R package

This package contains dimensionality functions presented in Nakamura et al 2019, beta-diversity measures presented in Nakamura et al. (2020), and some other function useful for quantify different dimensions of biodiversity.

To install this package use:

devtools::install_github("GabrielNakamura/Dimensionality_package")

The Dimensionality package is under development.
